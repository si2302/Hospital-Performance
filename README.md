# Hospital-Performance
📋 Project Overview
This project is a comprehensive Healthcare Analytics Dashboard designed to provide hospital administrators with actionable insights into patient demographics, treatment costs, and operational efficiency. By visualizing key performance indicators (KPIs), this tool helps in optimizing resource allocation and understanding financial trends across different departments.

🚀 Key Features & Insights
Executive KPIs: High-level overview of Total Patients (1K), Average Treatment Cost ($38.10K), and Average Length of Stay (7.96 days).

Departmental Analysis: Breakdown of costs and stay duration across Neurology, General, Ortho, and Cardiology.

Physician Performance: Tracking treatment costs and patient distribution by individual doctors (Dr. John, Dr. Patel, etc.).

Financial Trends: Monthly "Sum of Treatment Cost" line chart to identify seasonal peaks in hospital revenue.

Demographic Segmentation: Gender-based cost analysis (53% Female / 47% Male) to understand patient profiles.

Interactive Filters: Dynamic slicing by Gender, Department, and Doctor Name for granular data exploration.
